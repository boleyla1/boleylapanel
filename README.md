# BoleylaPanel

A comprehensive VPN panel management system built with FastAPI and modern web technologies.

## Features

- Multi-protocol support (VLESS, Reality, Trojan, Hysteria2, OpenVPN, L2TP)
- Node management system
- User management with JWT authentication
- Real-time traffic monitoring
- Automated backup system
- Professional dashboard with analytics

## Tech Stack

- **Backend:** Python 3.11, FastAPI, SQLAlchemy 2.0
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5.3 RTL
- **Database:** MySQL 8.0
- **Deployment:** Docker, Docker Compose, Nginx

## Requirements

- Docker 20.10+
- Docker Compose 2.0+
- 2GB RAM minimum
- 10GB disk space
